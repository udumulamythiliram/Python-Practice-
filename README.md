## Setup
For the currency converter, get a free API key at exchangerate-api.com and replace "your_api_key_here" in the code with your key.
## How to run tests
1. Install pytest: `pip install pytest`
2. Navigate to the project folder
3. Run: `pytest test_bmi.py`
